# Trace

Trace is a mobile app that guesses which famous person a user is thinking of by asking a short series of questions.

## Planned structure

- `apps/mobile` — React Native / Expo app
- `services/api` — backend API
- `packages/engine` — ranking and question-selection logic
- `data/questions` — question bank JSON
- `data/seeds` — seed entity data
- `data/eval` — evaluation cases and benchmarks
- `docs` — product and technical notes
